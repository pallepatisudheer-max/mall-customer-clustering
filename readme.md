# Mall Customer Clustering

A Machine Learning project that segments mall customers using K-Means Clustering.

## Features

- Data Preprocessing
- Elbow Method for optimal clusters
- K-Means Clustering
- Customer Segmentation Visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Structure

```
main.py
data_preprocessing.py
cluster_analysis.py
segmentation.py
pca_visualization.py
mall_customers.csv
requirements.txt
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python main.py
```

## Dataset

Mall Customers dataset containing:
- CustomerID
- Gender
- Age
- Annual Income
- Spending Score
